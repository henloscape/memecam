# Miui Camera for prebuilt vendors ###

- ### What is this ? ### 

  This is a MIUI camera that can be prebuilt to your ROM. 

- ### How to use ###

  Include this in your device makefile : 

  ```  $(call inherit-product, vendor/xiaomi/miuicamera/config.mk ```

  Continue your build.
